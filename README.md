# Jupyter Survey

This creates a survey form using ipywidgets and collects user responses in a dataframe called "df.csv" when the user clicks "Submit". If run in JupyterLab, the .csv file will appear as another file in the lab environment, right next to the .ipynb and the requirements.txt files. 

To see how the survey looks with Viola, click the following Binder link: https://notebooks.gesis.org/binder/jupyter/user/mkinlan-jupyter_survey-au26wyq7/voila/render/survey_widgets.ipynb?token=LTxZKs60RBaphRiN2ESTpw
